# FROM — Survive &amp; Solve

A browser-based mystery-survival game inspired by the TV show *FROM*. Built as an unofficial fan tribute and portfolio piece.

Solve a mystery by day, survive the monsters by night. Each level plays differently.

**Live:** enable GitHub Pages (below) and it runs at `https://nikku2204.github.io/FROM-game/`

## Play

- **Menu** — pick a level. Levels unlock as you survive the one before.
- **Level 1 — Boyd.** Daylight for only three conversations but four townsfolk. Gather clues about the symbol carved into the trees, decide who to trust (one of them is lying), then use what you learned to assemble the talisman and survive the night.
- **Sound** — original eerie ambience, synthesized in the browser. Toggle it bottom-right.

No API key, no backend, no login — it's all static and runs entirely in the browser.

## Structure

```
FROM-game/
├── index.html              the shell: styles + page + loads the scripts
├── engine.js               shared systems: audio, backgrounds, notebook,
│                           the level menu, save/progress
├── levels/
│   └── level1-boyd.js       Boyd's level (registers itself with the engine)
└── assets/                  character portraits, locations, monsters
```

## Adding a level

1. Create `levels/level2-tabitha.js`. At the bottom it registers itself:
   ```js
   Engine.addLevel({ id:"tabitha", num:2, title:"Tabitha",
                     tagline:"Remember, before the dark does", start: bootTabitha });
   ```
   Inside, draw your screens with the shared API — `Engine.setBg(...)`, `Engine.render(html)`,
   `Engine.notebook`, `Engine.danger(...)`, `Engine.nightClass(...)` — and the level can play
   however you like. Call `Engine.complete("tabitha")` when the player survives.
2. Add one line to `index.html`:
   ```html
   <script src="levels/level2-tabitha.js"></script>
   ```
3. Done — it shows up in the menu and unlocks after the previous level.

## Host it (GitHub Pages)

1. Upload these files to the repo root (keep the `levels/` and `assets/` folders).
2. **Settings → Pages → Build and deployment → Deploy from a branch → `main` / `root` → Save.**
3. Wait ~1 minute, then visit `https://nikku2204.github.io/FROM-game/`.

Paths are all relative, so it works in the repo's subfolder.

## Art &amp; audio

All imagery was generated by the author as original art (via Gemini) to evoke the show's world — no real actors, stills, or copyrighted assets. The music is original, synthesized live. *FROM* is the property of its creators; this is an unofficial fan tribute, not affiliated with or endorsed by them.

## Tech

Vanilla HTML/CSS/JavaScript — no frameworks. Web Audio API for sound. Progress saved via `localStorage` where available. Responsive; respects `prefers-reduced-motion`.
